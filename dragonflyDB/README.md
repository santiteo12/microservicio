# Dragonfly
## Estructura para la carpeta Dragonfly
```
carpeta_del_usuario
├── dragonfly
│   └── data # Carpeta con la configuración de Dragonfly
│   └── docker-compose.yml
│   └── .env # Renombrar .env-example y cambiar los valores de las variables de entorno
```
## Ejecución de Dragonfly

desde la terminal en la carpeta (dragonflyDB) ejecutar:
```
docker compose up
```
